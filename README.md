# College Schedule 📚

Простое и удобное приложение для просмотра расписания пар и звонков в колледже.

## Функциональность

- 📅 **Расписание на сегодня** — быстрый просмотр пар на текущий день
- 📖 **Расписание на неделю** — полная таблица на всю неделю
- 🔔 **Расписание звонков** — время начала и конца каждой пары
- 📂 **Импорт файлов** — поддержка Excel (.xls, .xlsx)

## Технологии

- **Python 3.13.2**
- **kivy** — библиотека для GUI
- **openpyxl** — работа с файлами .xlsx
- **pandas** — чтение и обработка данных из Excel
- **xlrd** — работа с файлами .xls
- **os** — работа с файлами и путями
- **sys** — работа с путями импорта
- **datetime** — получение текущего дня недели

## Установка

1. Клонируйте репозиторий:
```bash
git clone https://github.com/angry321/college-schedule.git
cd college-schedule
```

2. Установите зависимости:
```bash
pip install -r requirements.txt
```

3. Запустите приложение:
```bash
python main.py
```

## Использование

1. При первом запуске нажмите кнопку **"Выбрать файл (.xls / .xlsx)"**
2. Выберите файл с расписанием колледжа
3. Используйте основные кнопки для просмотра:
   - **Расписание на сегодня** — какие пары сегодня
   - **Расписание на неделю** — полное расписание на неделю
   - **Расписание звонков** — время начала и конца пар

Путь к файлу сохраняется автоматически в `path_to_table.txt`

## Структура проекта

```
college-schedule/

│   └── calls.py               # Расписание звонков

├── path_to_table.txt          # Конфиг с абсолютным путём до таблицы (создаётся автоматически)

```
