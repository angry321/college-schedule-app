# College Schedule App 📚

Простое мобильное приложение для просмотра расписания пар и звонков в колледже.

## Функциональность

- 📅 **Расписание на сегодня** — быстрый просмотр пар на текущий день
- 📖 **Расписание на неделю** — полная таблица на всю неделю
- 🔔 **Расписание звонков** — время начала и конца каждой пары
- 📂 **Импорт таблиц** — поддержка Excel таблиц

# Технологии
## Основные

- **Python 3.13.2**
- **FastAPI**
- **React Native (Expo)**
- **TypeScript**
- **JavaScript**

## Библиотеки / инструменты
- **expo-document-picker** - выбор Excel файла
- **pandas** - обработка Excel
- **openpyxl / xlrd** - чтение таблиц

## Установка
# 1. Клонируйте репозиторий:
```bash
git clone https://github.com/angry321/college-schedule-app.git
cd college-schedule-app/mobile
```

# 2. Установите зависимости:
```bash
npm install
```

# 3. Запустите приложение:
```bash
npx expo start
```

## Сборка APK (Android)
1. Регистрация
Для сборки приложения нужен аккаунт Expo:
- **1. Перейти на сайт https://expo.dev**
- **2. Зарегестрироваться**
- **3. Войти в аккаунт в терминале:**
```bash
npx expo login
```

2. Установите зависимости:
Перейти в папку мобильного приложения:
```bash
cd mobile
```
Установить зависимости:
```bash
npm install
```

3. Установка EAS CLI
EAS - инструмент для сборки APK:
```bash
npm install -g eas-cli
```
Проверка:
```bash
eas --version
```

4. Инициализация проекта
Выполняется один раз:
```bash
eas init
```

5. Сборка APK
Для получения APK-файла:
```bash
eas build -p android --profile preview
```

6. Ожидание сборки
- **Сборка занимает ~5-20 минут**
- **После завершения появится ссылка**
- **Скачать APK-файл с сайта**

7. Установить на телефон
