# College Schedule App 📚

Простое мобильное приложение для просмотра расписания пар и звонков в колледже.

## Функциональность

- 📅 **Расписание на сегодня** — быстрый просмотр пар на текущий день
- 📖 **Расписание на неделю** — полная таблица на всю неделю
- 🔔 **Расписание звонков** — время начала и конца каждой пары
- 📂 **Импорт таблиц** — поддержка Excel таблиц

# Технологии
## Основные

- **Python 3.13.2**
- **FastAPI**
- **React Native (Expo)**
- **TypeScript**
- **JavaScript**

## Библиотеки / инструменты
- **expo-document-picker** - выбор Excel файла
- **pandas** - обработка Excel
- **openpyxl / xlrd** - чтение таблиц

## Установка
# 1. Клонируйте репозиторий:
```bash
git clone https://github.com/angry321/college-schedule-app.git
cd college-schedule-app/mobile
```

# 2. Установите зависимости:
```bash
npm install
```

# 3. Запустите приложение:
```bash
npx expo start
```

## Сборка APK (Android)
# 1. Регистрация
   Для сборки приложения нужен аккаунт Expo:
1. Перейти на сайт https://expo.dev
2. Зарегестрироваться
3. Войти в аккаунт в терминале:
```bash
npx expo login
```

# 2. Установите зависимости:
   Перейти в папку мобильного приложения:
```bash
cd mobile
```
Установить зависимости:
```bash
npm install
```

# 3. Установка EAS CLI
   EAS - инструмент для сборки APK:
```bash
npm install -g eas-cli
```
Проверка:
```bash
eas --version
```

# 4. Инициализация проекта
   Выполняется один раз:
```bash
eas init
```

# 5. Сборка APK
   Для получения APK-файла:
```bash
eas build -p android --profile preview
```

# 6. Ожидание сборки


## Использование

1. При первом запуске нажмите кнопку **"Выбрать файл (.xls / .xlsx)"**
2. Выберите файл с расписанием колледжа
3. Используйте основные кнопки для просмотра:
   - **Расписание на сегодня** — какие пары сегодня
   - **Расписание на неделю** — полное расписание на неделю
   - **Расписание звонков** — время начала и конца пар

Путь к файлу сохраняется автоматически в `path_to_table.txt`

## Структура проекта

```
college-schedule/
├── main.py                    # Главное окно приложения
├── config.py
├── schedule/
│   ├── lessons_for_today.py   # Логика для расписания на день
│   ├── lessons_for_week.py    # Логика для расписания на неделю
│   └── calls.py               # Расписание звонков
├── requirements.txt           # Зависимости проекта
├── path_to_table.txt          # Конфиг с абсолютным путём до таблицы (создаётся автоматически)
└── README.md
```
