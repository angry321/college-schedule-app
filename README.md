# College Schedule App 📚

Простое мобильное приложение для просмотра расписания пар и звонков в колледже.

---

## Функциональность

- 📅 **Расписание на сегодня** — быстрый просмотр пар на текущий день
- 📖 **Расписание на неделю** — полная таблица на всю неделю
- 🔔 **Расписание звонков** — время начала и конца каждой пары
- 📂 **Импорт таблиц** — поддержка Excel файлов (.xlsx / .xls)

## Установка и запуск

### 1. Клонируйте репозиторий

```bash
git clone https://github.com/angry321/college-schedule-app.git
cd college-schedule-app/mobile
```

### 2. Установите зависимости

```bash
npm install
```

### 3. Запустите приложение

```bash
npx expo start
```

После этого откроется Expo DevTools. Отсканируйте QR-код через приложение **Expo Go** на телефоне или запустите эмулятор.

---

## Сборка APK (Android)

### 1. Регистрация на Expo

Для сборки нужен аккаунт на [expo.dev](https://expo.dev):

```bash
npx expo login
```

### 2. Перейдите в папку мобильного приложения

```bash
cd mobile
npm install
```

### 3. Установите EAS CLI

```bash
npm install -g eas-cli
eas --version  # проверка установки
```

### 4. Инициализация проекта (один раз)

```bash
eas init
```

### 5. Сборка APK

```bash
eas build -p android --profile preview
```

### 6. Получение файла

- Сборка занимает **5–20 минут**
- После завершения появится ссылка на скачивание
- Скачайте APK и установите на телефон

> ⚠️ При установке APK вручную может потребоваться разрешить установку из неизвестных источников в настройках Android.
